## 🌟 Overview

**CODEX** is an interactive, single-page curriculum website built with vanilla HTML, CSS, and JavaScript. It presents a structured learning path across two progressive tracks:

- **CS Core** (22 Weeks) — Foundation to Junior/Associate SWE
- **CS Pro** (22 Additional Weeks) — Advanced to FAANG SDE-1

The page features premium dark-mode aesthetics, smooth scroll animations, a timeline-based curriculum layout, and a company marquee showcasing target employers.

---

## ✨ Features

- 🎨 **Premium Dark UI** — Custom design system with glassmorphism effects and gradient accents
- ✨ **Scroll Animations** — Intersection Observer-powered reveal animations
- 📊 **Phase Timeline** — 12-phase curriculum displayed as an interactive vertical timeline
- 🏢 **Company Marquee** — Auto-scrolling carousel of target companies (Google, Meta, Amazon, etc.)
- 📅 **Weekly Breakdown** — Detailed table showing week-by-week topics and deliverables
- 🗺️ **Student Roadmap** — Visual journey from enrollment to FAANG graduation
- 📱 **Responsive Design** — Mobile-friendly layout with adaptive grid systems
- 🔤 **Custom Typography** — Google Fonts (Syne, JetBrains Mono, DM Sans)
- 🎯 **Sticky Navigation** — Fixed nav bar with backdrop blur and smooth scroll links

## 📚 Curriculum Tracks

### 📘 CS Core — Weeks 1–22
| Phase | Topic | Weeks |
|-------|-------|-------|
| 01 | Programming Foundations & CS Basics | 1–3 |
| 02 | DSA — Arrays, Strings & Linear Structures | 4–7 |
| 03 | Frontend — React & Modern UI | 8–11 |
| 04 | Backend — Node.js, Express & Databases | 12–16 |
| 05 | DevOps Basics — Docker & Cloud Intro | 17–19 |
| 06 | CS Core Capstone & Interview Prep | 20–22 |

### 🔥 CS Pro — Weeks 23–44
| Phase | Topic | Weeks |
|-------|-------|-------|
| 07 | Advanced DSA — Trees, Graphs & DP | 23–27 |
| 08 | System Design — Fundamentals to Advanced | 28–31 |
| 09 | Cloud Architecture — AWS / GCP Advanced | 32–35 |
| 10 | AI / ML Engineering Integration | 36–39 |
| 11 | FAANG Interview Intensive | 40–42 |
| 12 | CS Pro Capstone & Launch | 43–44 |

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No build tools or dependencies required!

### Run Locally

```bash
# Clone the repository
git clone https://github.com/akhil-06/lpu_git.git

# Navigate into the project
cd lpu_git

# Open in your browser
open git_understanding/index.html
# or on Windows:
start git_understanding/index.html
```

Or simply open `git_understanding/index.html` directly in your browser.
